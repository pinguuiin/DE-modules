# DE-modules

This repository stores the learning modules and homework for the course Data Engineering Zoomcamp. The goal of the course is to master the fundamentals of data engineering and be able to build production-ready data pipelines from scratch.

## Modules
### Module 1: Containerization and Infrastructure as Code
Introduction to GCP • Docker and Docker Compose • Running PostgreSQL with Docker • Infrastructure setup with Terraform

### Module 2: Workflow Orchestration
Data Lakes and Workflow Orchestration • Workflow orchestration with Airflow

*The course is in progress and the contents will be updated accordingly.*
